# Marketing Analytics & Attribution Framework

## Overview
This project analyzes multi-channel marketing performance across Email, Social Media, Paid Search, and Display Ads using key KPIs and business metrics.

## Objectives
- Evaluate marketing channel effectiveness
- Calculate KPIs such as CTR, CPC, CAC, and ROAS
- Compare channel performance
- Provide data-driven marketing recommendations

## Tools Used
- Google Sheets / Excel
- Data Visualization (Charts)
- Basic Analytical Techniques

## Key Insights
- Email marketing is the most efficient channel (highest ROAS)
- Paid Search generates the highest revenue
- Display Ads show the lowest performance
- Social Media provides moderate engagement

## Recommendations
- Increase investment in Email campaigns
- Optimize Paid Search efficiency
- Reduce or improve Display Ads targeting
- Improve Social Media conversion strategy
